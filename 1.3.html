1. Forms of Hydrogen Wave Functions

The wave function is characterized by three quantum numbers:

Principal quantum number (): Determines energy levels.

Azimuthal quantum number (): Defines orbital shape.

Magnetic quantum number (): Specifies orientation in space.

(a) Radial Wave Function

Radial wave functions for the first few hydrogen orbitals:

1s Orbital ():

R_{1,0}(r) = \frac{2}{a_0^{3/2}} e^{-r/a_0}

2s Orbital ():

R_{2,0}(r) = \frac{1}{\sqrt{2} a_0^{3/2}} \left(2 - \frac{r}{a_0}\right) e^{-r/2a_0}

2p Orbital ():

R_{2,1}(r) = \frac{1}{\sqrt{6} a_0^{3/2}} \frac{r}{a_0} e^{-r/2a_0}

where is the Bohr radius.

(b) Angular Wave Function

s-Orbitals (): Spherically symmetric, no angular dependence.

p-Orbitals (): Have distinct lobes along different axes.

d-Orbitals (): More complex shapes (cloverleaf-like).
